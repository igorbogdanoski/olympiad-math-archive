–ü—Ä–≤–æ –≥–æ —Ä–∞–∑–≥–ª–µ–¥—É–≤–∞–º–µ $\triangle BAE$. –î–∞–¥–µ–Ω–æ –µ –¥–µ–∫–∞ $\overline{BE} = \overline{AB}$, —à—Ç–æ –∑–Ω–∞—á–∏ –¥–µ–∫–∞ —Ç—Ä–∏–∞–≥–æ–ª–Ω–∏–∫–æ—Ç –µ —Ä–∞–º–Ω–æ–∫—Ä–∞–∫ —Å–æ –æ—Å–Ω–æ–≤–∞ $AE$. –ê–≥–ª–∏—Ç–µ –ø—Ä–∏ –æ—Å–Ω–æ–≤–∞—Ç–∞ —Å–µ –µ–¥–Ω–∞–∫–≤–∏:

–í—Ç–æ—Ä–æ –≥–æ —Ä–∞–∑–≥–ª–µ–¥—É–≤–∞–º–µ $\triangle CAD$. –î–∞–¥–µ–Ω–æ –µ –¥–µ–∫–∞ $\overline{CD} = \overline{AC}$, —à—Ç–æ –∑–Ω–∞—á–∏ –¥–µ–∫–∞ —Ç—Ä–∏–∞–≥–æ–ª–Ω–∏–∫–æ—Ç –µ —Ä–∞–º–Ω–æ–∫—Ä–∞–∫ —Å–æ –æ—Å–Ω–æ–≤–∞ $AD$. –ê–≥–ª–∏—Ç–µ –ø—Ä–∏ –æ—Å–Ω–æ–≤–∞—Ç–∞ —Å–µ –µ–¥–Ω–∞–∫–≤–∏:

<summary>–ß–µ–∫–æ—Ä 3: –ö–æ—Ä–∏—Å—Ç–µ—ö–µ –Ω–∞ —Å–≤–æ—ò—Å—Ç–≤–æ—Ç–æ –Ω–∞ –ø—Ä–µ–∫–ª–æ–ø—É–≤–∞—ö–µ –∫–∞—ò —Ç–µ–º–µ—Ç–æ A</summary>

$$\angle BAE + \angle CAD = \angle BAC + \angle DAE$$

–ó–∞–º–µ–Ω—É–≤–∞–º–µ —Å–æ –ø–æ–∑–Ω–∞—Ç–∏—Ç–µ –≤—Ä–µ–¥–Ω–æ—Å—Ç–∏:

$$180^\circ - \frac{\beta + \gamma}{2} = 90^\circ + \angle DAE$$

$$180^\circ - \frac{90^\circ}{2} = 90^\circ + \angle DAE$$

$$180^\circ - 45^\circ = 90^\circ + \angle DAE$$

$$135^\circ = 90^\circ + \angle DAE$$

$$\angle DAE = 135^\circ - 90^\circ = 45^\circ$$

from manim import *
import numpy as np

class SolutionScene(Scene):
    def construct(self):
        self.camera.background_color = WHITE
        
        # --- CONFIGURATION (Right Triangle 3-4-5) ---
        # Vertex A is at Origin (90 degrees)
        A = np.array()
        C = np.array()
        B = np.array()
        
        # BC is hypotenuse, length 5
        # Direction vector B to C
        vec_bc = C - B
        unit_bc = vec_bc / np.linalg.norm(vec_bc)
        
        # E is on BC such that BE = AB = 3
        E = B + 3 * unit_bc

        # Direction C to B
        unit_cb = (B - C) / np.linalg.norm(B - C)
        D = C + 4 * unit_cb

        line_ad = Line(A, D, color=BLUE, stroke_width=3)
        line_ae = Line(A, E, color=RED, stroke_width=3)

        ra = RightAngle(Line(A, B), Line(A, C), length=0.3, color=BLACK)

        dot_a = Dot(A, color=BLACK)
        dot_b = Dot(B, color=BLACK)

        dot_d = Dot(D, color=BLUE)
        dot_e = Dot(E, color=RED)

        # Result formula
        res = MathTex("\\angle DAE = 45^\\circ", color=PURPLE).to_edge(DOWN)
        
        # Add to scene
        self.add(tri_abc, line_ad, line_ae, ra, angle_dae)
        self.add(dot_a, dot_b, dot_c, dot_d, dot_e)
        self.add(lbl_A, lbl_B, lbl_C, lbl_D, lbl_E, val_45, res)
