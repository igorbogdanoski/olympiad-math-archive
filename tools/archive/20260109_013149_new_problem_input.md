## üìê –î–µ—Ç–∞–ª–Ω–æ –†–µ—à–µ–Ω–∏–µ

<details>
<summary>–ß–µ–∫–æ—Ä 1: –û–∑–Ω–∞—á—É–≤–∞—ö–µ –Ω–∞ –∞–≥–ª–∏—Ç–µ –∏ —Å–≤–æ—ò—Å—Ç–≤–∞ –Ω–∞ —Ä–∞–º–Ω–æ–∫—Ä–∞–∫–∏—Ç–µ —Ç—Ä–∏–∞–≥–æ–ª–Ω–∏—Ü–∏</summary>

–ù–µ–∫–∞ –∞–≥–ª–∏—Ç–µ –Ω–∞ $\triangle ABC$ —Å–µ $\angle BAC = \alpha$ –∏ $\angle ABC = \beta$. –ë–∏–¥–µ—ò—ú–∏ —Ç—Ä–∏–∞–≥–æ–ª–Ω–∏–∫–æ—Ç –µ –ø—Ä–∞–≤–æ–∞–≥–æ–ª–µ–Ω –∫–∞—ò $C$, –∏–º–∞–º–µ $\alpha + \beta = 90^\circ$.

–†–∞–∑–≥–ª–µ–¥—É–≤–∞–º–µ $\triangle ACD$. –î–∞–¥–µ–Ω–æ –µ $\overline{AD} = \overline{AC}$, —à—Ç–æ –∑–Ω–∞—á–∏ –¥–µ–∫–∞ —Ç—Ä–∏–∞–≥–æ–ª–Ω–∏–∫–æ—Ç –µ —Ä–∞–º–Ω–æ–∫—Ä–∞–∫ —Å–æ –æ—Å–Ω–æ–≤–∞ $CD$. –ê–≥–ª–∏—Ç–µ –ø—Ä–∏ –æ—Å–Ω–æ–≤–∞—Ç–∞ —Å–µ –µ–¥–Ω–∞–∫–≤–∏:

$$\angle ACD = \angle ADC = \frac{180^\circ - \alpha}{2} = 90^\circ - \frac{\alpha}{2}$$

–†–∞–∑–≥–ª–µ–¥—É–≤–∞–º–µ $\triangle BCE$. –î–∞–¥–µ–Ω–æ –µ $\overline{BE} = \overline{BC}$, —à—Ç–æ –∑–Ω–∞—á–∏ –¥–µ–∫–∞ —Ç—Ä–∏–∞–≥–æ–ª–Ω–∏–∫–æ—Ç –µ —Ä–∞–º–Ω–æ–∫—Ä–∞–∫ —Å–æ –æ—Å–Ω–æ–≤–∞ $CE$. –ê–≥–ª–∏—Ç–µ –ø—Ä–∏ –æ—Å–Ω–æ–≤–∞—Ç–∞ —Å–µ –µ–¥–Ω–∞–∫–≤–∏:

$$\angle BCE = \angle BEC = \frac{180^\circ - \beta}{2} = 90^\circ - \frac{\beta}{2}$$

</details>

<details>
<summary>–ß–µ–∫–æ—Ä 2: –ê–Ω–∞–ª–∏–∑–∞ –Ω–∞ –ø—Ä–µ–∫–ª–æ–ø–æ—Ç –∫–∞—ò —Ç–µ–º–µ—Ç–æ C</summary>

–î–∞ –∑–∞–±–µ–ª–µ–∂–∏–º–µ –¥–µ–∫–∞ –±–∞—Ä–∞–Ω–∏–æ—Ç –∞–≥–æ–ª $\angle DCE$ –µ –¥–µ–ª –æ–¥ –¥–≤–∞—Ç–∞ –∞–≥–ª–∏ $\angle ACD$ –∏ $\angle BCE$. –í–∫—É–ø–Ω–∏–æ—Ç –∞–≥–æ–ª –∫–∞—ò —Ç–µ–º–µ—Ç–æ $C$ –µ $90^\circ$.

–ú–æ–∂–µ–º–µ –¥–∞ –≥–∏ –∏–∑—Ä–∞–∑–∏–º–µ –∞–≥–ª–∏—Ç–µ –∫–æ–∏ —Å–µ "–Ω–∞–¥–≤–æ—Ä" –æ–¥ $\angle DCE$:
- $\angle ACE = \angle ACB - \angle BCE = 90^\circ - (90^\circ - \frac{\beta}{2}) = \frac{\beta}{2}$
- $\angle BCD = \angle ACB - \angle ACD = 90^\circ - (90^\circ - \frac{\alpha}{2}) = \frac{\alpha}{2}$

–¢–æ–≥–∞—à, –∞–≥–æ–ª–æ—Ç $\angle DCE$ —Å–µ –ø—Ä–µ—Å–º–µ—Ç—É–≤–∞ –∫–∞–∫–æ:

$$\angle DCE = \angle ACB - \angle ACE - \angle BCD$$

$$\angle DCE = 90^\circ - \frac{\beta}{2} - \frac{\alpha}{2}$$

</details>

<details>
<summary>–ß–µ–∫–æ—Ä 3: –§–∏–Ω–∞–ª–Ω–∞ –ø—Ä–µ—Å–º–µ—Ç–∫–∞</summary>

–ì–æ —Å—Ä–µ–¥—É–≤–∞–º–µ –∏–∑—Ä–∞–∑–æ—Ç:

$$\angle DCE = 90^\circ - \left( \frac{\alpha + \beta}{2} \right)$$

–ë–∏–¥–µ—ò—ú–∏ $\alpha + \beta = 90^\circ$ (–∞–≥–ª–∏ –≤–æ –ø—Ä–∞–≤–æ–∞–≥–æ–ª–µ–Ω —Ç—Ä–∏–∞–≥–æ–ª–Ω–∏–∫), –∑–∞–º–µ–Ω—É–≤–∞–º–µ:

$$\angle DCE = 90^\circ - \frac{90^\circ}{2}$$

$$\angle DCE = 90^\circ - 45^\circ = 45^\circ$$

from manim import *
import numpy as np

class SolutionScene(Scene):
    def construct(self):
        self.camera.background_color = WHITE
        
        # --- CONFIGURATION (Right Triangle 3-4-5) ---
        C = ORIGIN
        A = np.array()
        B = np.array()
        
        # Triangle ABC
        tri_abc = Polygon(A, B, C, color=BLACK, stroke_width=4)
        
        # Calculate hypotenuse AB direction
        vec_ab = B - A
        len_ab = 5
        unit_ab = vec_ab / len_ab
        
        # D is on AB such that AD = AC = 4
        D = A + 4 * unit_ab
        
        # E is on AB such that BE = BC = 3
        # E is B + 3 * (-unit_ab)
        E = B - 3 * unit_ab
        
        # Lines from C to D and E
        line_cd = Line(C, D, color=BLUE, stroke_width=3)
        line_ce = Line(C, E, color=RED, stroke_width=3)
        
        # Points and Labels
        dot_a = Dot(A, color=BLACK)
        dot_b = Dot(B, color=BLACK)
        dot_c = Dot(C, color=BLACK)
        dot_d = Dot(D, color=BLUE)
        dot_e = Dot(E, color=RED)
        
        lbl_A = MathTex("A", color=BLACK).next_to(A, UP)
        lbl_B = MathTex("B", color=BLACK).next_to(B, RIGHT)
        lbl_C = MathTex("C", color=BLACK).next_to(C, DL)
        lbl_D = MathTex("D", color=BLUE).next_to(D, UR, buff=0.1)
        lbl_E = MathTex("E", color=RED).next_to(E, UR, buff=0.1)
        
        # Angle indicator for DCE
        ang_dce = Angle(line_ce, line_cd, radius=0.7, color=PURPLE)
        lbl_45 = MathTex("45^\\circ", color=PURPLE).scale(0.8).move_to(ang_dce.get_center() + [0.4, 0.4, 0])
        
        # Adding components
        self.add(tri_abc, line_cd, line_ce)
        self.add(dot_a, dot_b, dot_c, dot_d, dot_e)
        self.add(lbl_A, lbl_B, lbl_C, lbl_D, lbl_E, ang_dce, lbl_45)
        
        # Highlighting side equalities with markers
        # No Cyrillic, just ticks or labels
        label_ac = MathTex("b", color=BLACK).next_to(Line(A,C).get_center(), LEFT)
        label_ad = MathTex("b", color=BLACK).next_to(Line(A,D).get_center(), UR, buff=-0.2)
        
        self.add(label_ac, label_ad)
