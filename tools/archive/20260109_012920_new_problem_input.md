–î–∞ –≥–∏ –æ–∑–Ω–∞—á–∏–º–µ –∞–≥–ª–∏—Ç–µ –Ω–∞ $\triangle ABC$ —Å–æ $\angle BAC = \alpha$, $\angle ABC = \beta$ –∏ $\angle ACB = \delta$. (–ö–æ—Ä–∏—Å—Ç–∏–º–µ $\delta$ –∑–∞ –¥–∞ –Ω–µ –≥–æ –º–µ—à–∞–º–µ —Å–æ –¥–∞–¥–µ–Ω–∏–æ—Ç $\gamma$).

–í–æ —Ç—Ä–∏–∞–≥–æ–ª–Ω–∏–∫–æ—Ç $\triangle ABD$, –∏–º–∞–º–µ $\overline{AB} = \overline{BD}$. –û–≤–æ—ò —Ç—Ä–∏–∞–≥–æ–ª–Ω–∏–∫ –µ —Ä–∞–º–Ω–æ–∫—Ä–∞–∫ —Å–æ –æ—Å–Ω–æ–≤–∞ $AD$. –ê–≥–ª–∏—Ç–µ –ø—Ä–∏ –æ—Å–Ω–æ–≤–∞—Ç–∞ —Å–µ –µ–¥–Ω–∞–∫–≤–∏:

$$\angle BAD = \angle BDA = \frac{180^\circ - \beta}{2} = 90^\circ - \frac{\beta}{2}$$

–í–æ —Ç—Ä–∏–∞–≥–æ–ª–Ω–∏–∫–æ—Ç $\triangle ACE$, –∏–º–∞–º–µ $\overline{AC} = \overline{CE}$. –û–≤–æ—ò —Ç—Ä–∏–∞–≥–æ–ª–Ω–∏–∫ –µ —Ä–∞–º–Ω–æ–∫—Ä–∞–∫ —Å–æ –æ—Å–Ω–æ–≤–∞ $AE$. –ê–≥–ª–∏—Ç–µ –ø—Ä–∏ –æ—Å–Ω–æ–≤–∞—Ç–∞ —Å–µ –µ–¥–Ω–∞–∫–≤–∏:

$$\angle CAE = \angle CEA = \frac{180^\circ - \delta}{2} = 90^\circ - \frac{\delta}{2}$$

<details>
<summary>–ß–µ–∫–æ—Ä 2: –í–æ—Å–ø–æ—Å—Ç–∞–≤—É–≤–∞—ö–µ –Ω–∞ –≤—Ä—Å–∫–∞—Ç–∞ –ø–æ–º–µ—ì—É –∞–≥–ª–∏—Ç–µ –∫–∞—ò —Ç–µ–º–µ—Ç–æ A</summary>

–î–∞ –≥–æ –Ω–∞–±—ô—É–¥—É–≤–∞–º–µ —Ç–µ–º–µ—Ç–æ $A$. –ë–∏–¥–µ—ò—ú–∏ —Ç—Ä–∏–∞–≥–æ–ª–Ω–∏–∫–æ—Ç –µ —Ç–∞–ø–æ–∞–≥–æ–ª–µ–Ω –∫–∞—ò $A$, —Ç–æ—á–∫–∏—Ç–µ –Ω–∞ —Å—Ç—Ä–∞–Ω–∞—Ç–∞ $BC$ —Å–µ —Ä–∞—Å–ø–æ—Ä–µ–¥–µ–Ω–∏ –≤–æ —Ä–µ–¥–æ—Å–ª–µ–¥ $B, E, D, C$. –ê–≥–æ–ª–æ—Ç $\angle DAE$ –µ –ø—Ä–µ–∫–ª–æ–ø–æ—Ç –Ω–∞ –∞–≥–ª–∏—Ç–µ $\angle BAD$ –∏ $\angle CAE$. –í–∞–∂–∏ —Ä–∞–≤–µ–Ω—Å—Ç–≤–æ—Ç–æ:

$$\alpha = \angle BAD + \angle CAE - \angle DAE$$

–ó–∞–º–µ–Ω—É–≤–∞—ò—ú–∏ –≥–∏ –∏–∑—Ä–∞–∑–∏—Ç–µ –æ–¥ –ß–µ–∫–æ—Ä 1 –∏ –¥–∞–¥–µ–Ω–∏–æ—Ç $\angle DAE = \gamma$, –¥–æ–±–∏–≤–∞–º–µ:

$$\alpha = \left( 90^\circ - \frac{\beta}{2} \right) + \left( 90^\circ - \frac{\delta}{2} \right) - \gamma$$

$$\alpha = 180^\circ - \frac{\beta + \delta}{2} - \gamma$$

<details>
<summary>–ß–µ–∫–æ—Ä 3: –ö–æ—Ä–∏—Å—Ç–µ—ö–µ –Ω–∞ –∑–±–∏—Ä–æ—Ç –Ω–∞ –∞–≥–ª–∏ –≤–æ ABC</summary>

–ì–æ –∑–∞–º–µ–Ω—É–≤–∞–º–µ –æ–≤–∞ –≤–æ —Ä–∞–≤–µ–Ω–∫–∞—Ç–∞ –æ–¥ –ø—Ä–µ—Ç—Ö–æ–¥–Ω–∏–æ—Ç —á–µ–∫–æ—Ä:

$$\alpha = 180^\circ - \left( 90^\circ - \frac{\alpha}{2} \right) - \gamma$$

$$\alpha = 90^\circ + \frac{\alpha}{2} - \gamma$$

–û–¥–∑–µ–º–∞–º–µ $\frac{\alpha}{2}$ –æ–¥ –¥–≤–µ—Ç–µ —Å—Ç—Ä–∞–Ω–∏:

$$\alpha = 180^\circ - 2\gamma$$

class SolutionScene(Scene):
