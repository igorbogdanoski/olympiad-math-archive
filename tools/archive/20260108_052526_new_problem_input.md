## üìê –î–µ—Ç–∞–ª–Ω–æ –†–µ—à–µ–Ω–∏–µ

<details>
<summary>–ß–µ–∫–æ—Ä 1: –ü–æ—Å—Ç–∞–≤—É–≤–∞—ö–µ –Ω–∞ —Ä–∞–≤–µ–Ω–∫–∞—Ç–∞ (–î–µ–ª –∞)</summary>

–î–∞–¥–µ–Ω–æ –µ $f(x) = ax + b$. –ì–æ –∑–∞–º–µ–Ω—É–≤–∞–º–µ –æ–≤–∞ –≤–æ –ª–µ–≤–∞—Ç–∞ —Å—Ç—Ä–∞–Ω–∞ –Ω–∞ —Ä–∞–≤–µ–Ω—Å—Ç–≤–æ—Ç–æ:
$$ f(b+\sqrt{2}) = a(b+\sqrt{2}) + b = ab + a\sqrt{2} + b $$
$$ f(b\sqrt{2} + 2) = a(b\sqrt{2} + 2) + b = ab\sqrt{2} + 2a + b $$

–ó–±–∏—Ä–æ—Ç –Ω–∞ –ª–µ–≤–∞—Ç–∞ —Å—Ç—Ä–∞–Ω–∞ –µ:
$$ LHS = (ab + a\sqrt{2} + b) + (ab\sqrt{2} + 2a + b) $$

$$ LHS = (ab + 2b + 2a) + (a + ab)\sqrt{2} $$

–î–µ—Å–Ω–∞—Ç–∞ —Å—Ç—Ä–∞–Ω–∞ –µ:
$$ RHS = b(a+2) - 2(2\sqrt{2}a - 1) = ab + 2b - 4a\sqrt{2} + 2 $$
$$ RHS = (ab + 2b + 2) - 4a\sqrt{2} $$
</details>

<details>
<summary>–ß–µ–∫–æ—Ä 2: –°–∏—Å—Ç–µ–º —Ä–∞–≤–µ–Ω–∫–∏ –∏ —Ä–µ—à–µ–Ω–∏–µ –∑–∞ a –∏ b</summary>

–ì–∏ –∏–∑–µ–¥–Ω–∞—á—É–≤–∞–º–µ $LHS$ –∏ $RHS$. –ë–∏–¥–µ—ò—ú–∏ $a, b \in \mathbb{Q}$, –∫–æ–µ—Ñ–∏—Ü–∏–µ–Ω—Ç–∏—Ç–µ –ø—Ä–µ–¥ $\sqrt{2}$ –º–æ—Ä–∞ –¥–∞ –±–∏–¥–∞—Ç –µ–¥–Ω–∞–∫–≤–∏, –∫–∞–∫–æ –∏ —Å–ª–æ–±–æ–¥–Ω–∏—Ç–µ —á–ª–µ–Ω–æ–≤–∏.

$$ a + ab = -4a $$
–ë–∏–¥–µ—ò—ú–∏ –µ –¥–∞–¥–µ–Ω–æ $a \neq 0$, –º–æ–∂–µ–º–µ –¥–∞ –ø–æ–¥–µ–ª–∏–º–µ —Å–æ $a$:
$$ 1 + b = -4 \implies b = -5 $$

2. –°–ø–æ—Ä–µ–¥—É–≤–∞—ö–µ –Ω–∞ —Ä–∞—Ü–∏–æ–Ω–∞–ª–Ω–∏—Ç–µ –¥–µ–ª–æ–≤–∏:
$$ ab + 2b + 2a = ab + 2b + 2 $$
–ü–æ –∫—Ä–∞—Ç–µ—ö–µ –Ω–∞ $ab + 2b$ –æ–¥ –¥–≤–µ—Ç–µ —Å—Ç—Ä–∞–Ω–∏:
$$ 2a = 2 \implies a = 1 $$

–ó–Ω–∞—á–∏, –±–∞—Ä–∞–Ω–∞—Ç–∞ —Ñ—É–Ω–∫—Ü–∏—ò–∞ –µ:
$$ \boxed{f(x) = x - 5} $$

<summary>–ß–µ–∫–æ—Ä 3: –ü–æ—Å—Ç–∞–≤—É–≤–∞—ö–µ –Ω–∞ –Ω–µ—Ä–∞–≤–µ–Ω—Å—Ç–≤–æ—Ç–æ (–î–µ–ª –±)</summary>

<details>
<summary>–ß–µ–∫–æ—Ä 4: –ì—Ä—É–ø–∏—Ä–∞—ö–µ –∏ –ø—Ä–∏–º–µ–Ω–∞ –Ω–∞ –Ω–µ—Ä–∞–≤–µ–Ω—Å—Ç–≤–æ</summary>

$$ S = (\sqrt{1} + \sqrt{49}) + (\sqrt{2} + \sqrt{48}) + \dots + (\sqrt{24} + \sqrt{26}) + \sqrt{25} $$
–ò–º–∞–º–µ 24 –ø–∞—Ä–æ–≤–∏ –∏ –µ–¥–µ–Ω —Å—Ä–µ–¥–µ–Ω —á–ª–µ–Ω ($\sqrt{25}$).

–ì–æ –∫–æ—Ä–∏—Å—Ç–∏–º–µ –Ω–µ—Ä–∞–≤–µ–Ω—Å—Ç–≤–æ—Ç–æ –º–µ—ì—É –∞—Ä–∏—Ç–º–µ—Ç–∏—á–∫–∞ –∏ –∫–≤–∞–¥—Ä–∞—Ç–Ω–∞ —Å—Ä–µ–¥–∏–Ω–∞ –∑–∞ –¥–≤–∞ –ø–æ–∑–∏—Ç–∏–≤–Ω–∏ –±—Ä–æ—ò–∞ $u, v$ ($u \neq v$):

–ê–∫–æ —Å—Ç–∞–≤–∏–º–µ $u = \sqrt{x}$ –∏ $v = \sqrt{y}$, –¥–æ–±–∏–≤–∞–º–µ:
$$ \sqrt{x} + \sqrt{y} < \sqrt{2(x+y)} $$

–ì–æ –ø—Ä–∏–º–µ–Ω—É–≤–∞–º–µ –æ–≤–∞ –Ω–∞ —Å–µ–∫–æ—ò –ø–∞—Ä $\sqrt{k} + \sqrt{50-k}$:
$$ \sqrt{k} + \sqrt{50-k} < \sqrt{2(k + 50 - k)} = \sqrt{2 \cdot 50} = \sqrt{100} = 10 $$

–ó–Ω–∞—á–∏, –∑–±–∏—Ä–æ—Ç –Ω–∞ —Å–µ–∫–æ—ò –ø–∞—Ä –µ —Å—Ç—Ä–æ–≥–æ –ø–æ–º–∞–ª –æ–¥ 10.

*   **–ü—Ä–∏–º–∞—Ä–Ω–∞ –≤–µ—à—Ç–∏–Ω–∞:** –°–ø–æ—Ä–µ–¥—É–≤–∞—ö–µ –∫–æ–µ—Ñ–∏—Ü–∏–µ–Ω—Ç–∏ (Comparing Coefficients).
*   **–ü–æ—Ç—Ä–µ–±–Ω–∏ –ø—Ä–µ–¥–∑–Ω–∞–µ—ö–∞:** –ò—Ä–∞—Ü–∏–æ–Ω–∞–ª–Ω–∏ –±—Ä–æ–µ–≤–∏, –ù–µ—Ä–∞–≤–µ–Ω—Å—Ç–≤–∞ (AM-QM –∏–ª–∏ –ö–æ—à–∏-–®–≤–∞—Ä—Ü).

# Manim Code
```python
from manim import *

class InequalityProof(Scene):
    def construct(self):
        self.camera.background_color = WHITE
        
        # Title
        title = Text("–î–æ–∫–∞–∑ –Ω–∞ –Ω–µ—Ä–∞–≤–µ–Ω—Å—Ç–≤–æ—Ç–æ", font_size=40, color=BLACK).to_edge(UP)
        self.play(Write(title))
        
        # Step 1: The Sum
        sum_tex = MathTex(
            "S = \\sqrt{1} + \\sqrt{2} + \\dots + \\sqrt{49}", 
            color=BLACK
        ).shift(UP)
        self.play(Write(sum_tex))
        self.wait(1)
        
        # Step 2: Pairing Strategy
        # Transform to paired version
        pairs_tex = MathTex(
            "S = (\\sqrt{1} + \\sqrt{49}) + (\\sqrt{2} + \\sqrt{48}) + \\dots + \\sqrt{25}",
            color=BLACK
        ).shift(UP)
        
        self.play(Transform(sum_tex, pairs_tex))
        self.wait(1)
        
        # Step 3: The Inequality Rule
        rule = MathTex(
            "\\sqrt{x} + \\sqrt{y} < \\sqrt{2(x+y)}",
            color=RED
        ).next_to(sum_tex, DOWN, buff=1)
        
        rule_box = SurroundingRectangle(rule, color=RED, buff=0.2)
        rule_label = Text("AM < QM", font_size=24, color=RED).next_to(rule_box, RIGHT)
        
        self.play(Write(rule), Create(rule_box), Write(rule_label))
        self.wait(1)
        
        # Step 4: Applying the rule
        # Example for first pair
        example = MathTex(
            "\\sqrt{1} + \\sqrt{49} < \\sqrt{2(1+49)} = \\sqrt{100} = 10",
            color=BLUE
        ).next_to(rule, DOWN, buff=0.5)
        
        self.play(Write(example))
        self.wait(1)
        
        # Step 5: Final Calculation
        final_calc = MathTex(
            "S < 24 \\times 10 + 5 = 245",
            color=BLACK
        ).next_to(example, DOWN, buff=1)
        
        conclusion = MathTex(
            "\\therefore S - 245 < 0",
            color=BLACK
        ).next_to(final_calc, DOWN)
        
        self.play(Write(final_calc))
        self.play(Write(conclusion))
        self.play(Indicate(conclusion, color=GREEN))
        
        self.wait(2)
```
