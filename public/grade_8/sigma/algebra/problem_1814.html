Геометриска прогресија и услови

Текст на задачата

Нека (a, b, c) се три последователни члена на геометриска прогресија со позитивни реални членови и сооднос (q \neq 1). Ако важи [ a + b + c = 21 \quad \text{и} \quad abc = 64, ] да се најдат (a, b, c).

Анализа

Бидејќи (a, b, c) се последователни членови на геометриска прогресија, можеме да ги запишеме во облик: [ a = \frac{b}{q}, \quad c = bq. ]

Со користење на дадените услови, се добива систем равенки кој може да се реши чекор по чекор.

Клучната идеја е да се искористи средниот член и симетријата на геометриската прогресија.

Решение (детално)

Ќе решаваме чекор по чекор.

Чекор 1: Претставување преку средниот член

Нека: [ a = \frac{b}{q}, \quad c = bq. ]

Чекор 2: Користење на производот

Од условот (abc = 64) добиваме: [ \frac{b}{q} \cdot b \cdot bq = b^3 = 64. ]

Оттука: [ b = 4. ]

Чекор 3: Користење на збирот

Од условот (a + b + c = 21): [ \frac{4}{q} + 4 + 4q = 21. ]

Го делиме со 4: [ \frac{1}{q} + 1 + q = \frac{21}{4}. ]

Односно: [ q + \frac{1}{q} = \frac{17}{4}. ]

Чекор 4: Решавање за (q)

Го множиме со (q): [ q^2 - \frac{17}{4}q + 1 = 0. ]

Го множиме со 4: [ 4q^2 - 17q + 4 = 0. ]

Решаваме квадратна равенка: [ q = \frac{17 \pm \sqrt{289 - 64}}{8} = \frac{17 \pm 15}{8}. ]

Оттука: [ q = 4 \quad \text{или} \quad q = \frac14. ]

Чекор 5: Наоѓање на членовите

  • Ако (q = 4), тогаш: [ a = 1,\quad b = 4,\quad c = 16. ]

  • Ако (q = \frac14), тогаш: [ a = 16,\quad b = 4,\quad c = 1. ]

Алтернативен пристап (олимписки)

Се користи фактот дека кај геометриска прогресија важи: [ b^3 = abc. ] Потоа, од збирот се добива равенка за (q + \frac{1}{q}), која се решава директно.

Заклучок

Решенијата се: [ \boxed{(a,b,c) = (1,4,16) \ \text{или} \ (16,4,1)}. ]

За тутори (опционално)

  • Предуслови: геометриска прогресија, квадратни равенки
  • Клучна идеја: централниот член ја носи информацијата
  • Типична грешка: мешање на аритметичка и геометриска прогресија
