<details>

$$ a + b = S \quad (\text{—Å—Ç—Ä–∞–Ω–∞ –Ω–∞ } Q) $$
$$ b + c = S $$
$$ c + d = S $$
$$ d + a = S $$
–û–¥ –æ–≤–∞ —Å–ª–µ–¥–∏ $a=c$ and $b=d$.

$$ (a+b)^2 = 2a^2 + 2b^2 + x^2 $$
$$ a^2 + 2ab + b^2 = 2a^2 + 2b^2 + (a-b)^2 $$
$$ a^2 + 2ab + b^2 = 2a^2 + 2b^2 + a^2 - 2ab + b^2 $$
$$ a^2 + 2ab + b^2 = 3a^2 - 2ab + 3b^2 $$
