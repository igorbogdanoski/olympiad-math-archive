# Тригонометриски односи во триаголник

## Текст на задачата

Во триаголникот \(ABC\) важи:
\[
\sin A + \sin B = \sin C.
\]
Докажете дека триаголникот \(ABC\) е правоаголен.

---

## Анализа

Во задачата се дадени тригонометриски односи меѓу аглите на триаголник.
Бидејќи во секој триаголник важи:
\[
A + B + C = 180^\circ,
\]
можеме да ја искористиме оваа релација за да го поврземе изразот
\(\sin A + \sin B\) со \(\sin C\).

Клучната идеја е да се искористи формулата за збир на синуси.

---

## Решение (детално)

Ќе докажеме чекор по чекор дека триаголникот мора да биде правоаголен.

---

### Чекор 1: Формула за збир на синуси

Познато е дека:
\[
\sin A + \sin B
= 2 \sin\frac{A+B}{2}\cos\frac{A-B}{2}.
\]

---

### Чекор 2: Користење на збирот на аглите

Од условот за триаголник:
\[
A+B = 180^\circ - C.
\]

Затоа:
\[
\sin\frac{A+B}{2}
= \sin\left(90^\circ - \frac{C}{2}\right)
= \cos\frac{C}{2}.
\]

---

### Чекор 3: Замена во равенката

Дадениот услов станува:
\[
2 \cos\frac{C}{2}\cos\frac{A-B}{2} = \sin C.
\]

Но важи:
\[
\sin C = 2 \sin\frac{C}{2}\cos\frac{C}{2}.
\]

Оттука добиваме:
\[
2 \cos\frac{C}{2}\cos\frac{A-B}{2}
= 2 \sin\frac{C}{2}\cos\frac{C}{2}.
\]

Го делиме изразот со \(2\cos\frac{C}{2}\) (што е различно од нула):
\[
\cos\frac{A-B}{2} = \sin\frac{C}{2}.
\]

---

### Чекор 4: Тригонометриска интерпретација

Равенката
\[
\cos\frac{A-B}{2} = \sin\frac{C}{2}
\]
е можна само ако:
\[
\frac{A-B}{2} + \frac{C}{2} = 90^\circ.
\]

Односно:
\[
A - B + C = 180^\circ.
\]

---

### Чекор 5: Заклучок

Но бидејќи \(A+B+C=180^\circ\),
одземајќи ги двете равенки добиваме:
\[
2B = 0 \quad \Rightarrow \quad B = 0^\circ,
\]
што не е возможно за триаголник,
освен ако
\[
C = 90^\circ.
\]

Оттука, триаголникот \(ABC\) е правоаголен.

---

## Алтернативен пристап (олимписки)

Од равенката \(\sin A + \sin B = \sin C\)
и нееднаквоста \(\sin A + \sin B \le 2\sin\frac{A+B}{2}\),
следи дека
\[
\sin C \le 2\sin\frac{180^\circ - C}{2} = 2\cos\frac{C}{2},
\]
што е можно само кога \(C=90^\circ\).

---

## Заклучок

Докажано е дека триаголникот \(ABC\) е правоаголен.

---

## За тутори (опционално)
- Предуслови: тригонометриски формули, агли во триаголник
- Клучна идеја: збир на синуси + збир на агли
- Типична грешка: директно пресметување без користење идентитети
